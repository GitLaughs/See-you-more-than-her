# Aurora Capture Tool — A1 摄像头拍照工具

通过 USB Type-C 连接 A1 开发板的 SC132GS 摄像头，进行实时预览和拍照。

## 功能

- 实时预览 1280×640 灰度摄像头画面
- 拍照保存两种格式：
  - **1280×640**: 原始灰度图（通用用途）
  - **640×480**: 中心裁剪灰度图（YOLOv8 训练集）
- Web 前端界面，支持键盘快捷键

## 使用方法

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 连接 A1 开发板

通过 USB Type-C 数据线连接 A1 开发板到 PC。确保开发板已开机且摄像头正常工作。

### 3. 启动拍照工具

```bash
python aurora_capture.py
```

参数说明：
- `--device 0`: 摄像头设备 ID（默认 0）
- `--output <dir>`: 拍照保存目录（默认 `../../data/yolov8_dataset/raw/images`）
- `--port 5000`: Web 服务端口（默认 5000）

### 4. 打开浏览器

访问 `http://localhost:5000`，可以看到实时预览画面。

### 5. 拍照

- 点击 **📷 拍照 1280×640** 按钮拍摄原始灰度图
- 点击 **📷 拍照 640×480** 按钮拍摄 YOLOv8 训练用裁剪图
- 快捷键：按 `1` 拍摄 1280×640，按 `2` 拍摄 640×480

## 技术说明

### 摄像头格式

SC132GS 传感器通过 USB Type-C 输出 1280×640 灰度视频流。工具使用 OpenCV 打开摄像头时，会尝试设置 FOURCC 为 `Y800`/`GREY` 以正确解析灰度格式（避免被解析为错误的比例和灰度）。

### 640×480 裁剪

从 1280×640 原图中心裁剪 640×480 区域。预览画面中绿色矩形框表示裁剪区域。

### 与 SDK pipeline 的关系

SDK 中 `pipeline_image.cpp` 会将给到模型的输入尺寸裁剪到 720×540，此工具的 640×480 裁剪是用于制作 YOLO 训练数据集，两者是不同的裁剪场景。
