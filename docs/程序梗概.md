# 人脸检测驱动小车 — 程序梗概

## 一、项目概述

本项目实现基于 SmartSens A1 开发板（SC132GS SoC）的人脸检测驱动小车系统。A1 开发板负责图像采集与 AI 推理，通过 GPIO UART 将运动控制指令发送给 WHEELTEC C50X 底盘（STM32F407），实现"检测到人脸就直行"的核心功能。

### 系统架构

```
┌─────────────────────────────────────────────────────┐
```

## 三、核心处理流程

```
每帧处理 (ProcessOnce):
  1. 采集灰度图像 (720×1280)
  2. 裁剪到 720×540，偏移 370 像素
  3. SCRFD 人脸检测 (640×480 输入, 置信度阈值 0.4)
  4. RPLidar 扫描 → 检查前方 ±30° 是否有障碍物 (<0.3m)
  5. 决策:
     - 检测到人脸 AND 前方无障碍 → 发送前进指令 (100 mm/s)
     - 否则 → 发送停止指令
  6. OSD 渲染检测框到视频输出
```

## 四、通信协议

### 4.1 A1 → STM32 控制帧 (11 字节)

```
字节  内容        说明
[0]   0x7B       帧头
[1]   Cmd        命令: 0x00=正常控制, 0xFF=IP帧
[2]   0x00       保留
[3]   Vx_H       X轴速度高字节 (mm/s, int16_t)
[4]   Vx_L       X轴速度低字节
[5]   Vy_H       Y轴速度高字节 (AKM车=0)
[6]   Vy_L       Y轴速度低字节
[7]   Vz_H       Z轴转向角速度高字节 (AKM车=前轮转角)
[8]   Vz_L       Z轴转向角速度低字节
[9]   BCC        校验 = XOR(byte[0]..byte[8])
[10]  0x7D       帧尾
```

**速度编码示例:**
- 前进 0.1 m/s: Vx=100 → `[0x00, 0x64]`
- 后退 0.2 m/s: Vx=-200 → `[0xFF, 0x38]`
- 停止: Vx=Vy=Vz=0

### 4.2 STM32 → A1 状态帧 (24 字节)

```
字节      内容          说明
[0]       0x7B         帧头
[1]       Stop_Flag    停止标志
[2-3]     Vx           实际X速度
[4-5]     Vy           实际Y速度
[6-7]     Vz           实际Z角速度
[8-9]     Accel_X      加速度X
[10-11]   Accel_Y      加速度Y
[12-13]   Accel_Z      加速度Z
[14-15]   Gyro_X       角速度X
[16-17]   Gyro_Y       角速度Y
[18-19]   Gyro_Z       角速度Z
[20-21]   Voltage      电池电压 (mV)
[22]      BCC          校验 = XOR(byte[0]..byte[21])
[23]      0x7D         帧尾
```

## 五、关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 前进速度 | 100 mm/s | 安全测试速度 |
| 障碍安全距离 | 0.3 m | RPLidar 前方检测 |
| 前方检测角度 | ±30° | 扇形安全区域 |
| 人脸置信度阈值 | 0.4 | SCRFD 模型 |
| UART 波特率 | 115200 | A1 ↔ STM32 |
| 控制帧发送频率 | ~20 Hz | 与 STM32 采样率匹配 |
| PI 控制参数 | Kp=300, Ki=300 | STM32 端电机控制 |

## 六、构建与部署

```bash
# Docker 容器内完整 EVB 构建并自动保存到时间戳目录
docker exec A1_Builder bash -lc "bash /app/scripts/build_complete_evb.sh --skip-ros"

# 开发迭代：仅增量编译 Demo
docker exec A1_Builder bash -lc "bash /app/scripts/build_incremental.sh sdk ssne_face_drive_demo"

# 在 A1 开发板上运行
cd /app_demo
./run.sh
```
