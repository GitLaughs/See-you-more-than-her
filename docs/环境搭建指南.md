# 环境搭建指南

本文档描述如何从零搭建编译环境，确保任何人都能复现 ROS2 工作区和 AI Demo 的编译。

## 1. 宿主机要求

| 项目 | 最低要求 |
|------|---------|
| 操作系统 | Windows 10/11 (amd64) 或 Linux |
| Docker | Docker Desktop 4.x (Windows) 或 Docker Engine 24+ (Linux) |
| Git | 2.30+ |
| 磁盘空间 | 约 20GB（镜像 + SDK + 编译缓存） |
| 内存 | 建议 8GB+（分配给 Docker 至少 4GB） |

## 2. 构建 Docker 镜像

```powershell
cd docker/
docker build -f Dockerfile -t a1-sdk-builder:latest .
```

镜像基于 Ubuntu 24.04 Noble，已包含：
- cmake、build-essential、gcc/g++
- ROS2 Jazzy（ros-jazzy-ros-base）
- colcon 构建工具
- rosdep 依赖管理

## 3. 启动容器

```powershell
cd docker/
docker compose up -d
docker compose ps   # 确认 A1_Builder 运行中
```

容器挂载关系：

| 宿主机 | 容器内 | 用途 |
|--------|--------|------|
| `./src` | `/app/src` | 源码（a1_ssne_ai_demo + ros2_ws） |
| `./data` | `/app/smartsens_sdk` | SmartSens A1 SDK |
| `./models` | `/app/models` | ONNX / m1model 模型文件 |
| `./output` | `/app/output` | 编译产物输出 |
| `./scripts` | `/app/scripts` | 编译脚本 |

## 4. 容器内 ROS2 环境配置

> **注意**：如果使用上面的 Dockerfile 构建镜像，ROS2 已预装。
> 以下步骤仅在手动配置或镜像未包含 ROS2 时使用。

### 4.1 运行自动安装脚本

```bash
docker exec -it A1_Builder bash /app/scripts/install_ros2_jazzy.sh
```

脚本会依次完成：
1. 切换 apt 源为清华镜像（加速下载）
2. 安装基础工具（curl、gpg、lsb-release、ca-certificates）
3. 添加 ROS2 GPG 密钥和清华 ROS2 软件源
4. 安装 `ros-jazzy-ros-base`、`python3-colcon-common-extensions`、`python3-rosdep`
5. 初始化 rosdep
6. 配置 `/etc/bash.bashrc` 自动加载 ROS2 环境

### 4.2 手动安装（逐步操作）

如需手动操作，以下是完整步骤：

```bash
# 进入容器
docker exec -it A1_Builder bash

# 1. 切换 Ubuntu apt 源到清华镜像
cat > /etc/apt/sources.list.d/ubuntu.sources << 'EOF'
Types: deb
URIs: https://mirrors.tuna.tsinghua.edu.cn/ubuntu/
Suites: noble noble-updates noble-backports
Components: main universe restricted multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg

Types: deb
URIs: https://mirrors.tuna.tsinghua.edu.cn/ubuntu/
Suites: noble-security
Components: main universe restricted multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg
EOF

# 2. 安装基础工具
apt-get update
apt-get install -y --no-install-recommends \
  curl gpg lsb-release ca-certificates software-properties-common

# 3. 添加 ROS2 GPG 密钥
curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.asc \
  | gpg --dearmor > /usr/share/keyrings/ros-archive-keyring.gpg

# 4. 添加 ROS2 清华镜像源
ARCH=$(dpkg --print-architecture)
echo "deb [arch=$ARCH signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] \
  https://mirrors.tuna.tsinghua.edu.cn/ros2/ubuntu noble main" \
  > /etc/apt/sources.list.d/ros2.list

# 5. 安装 ROS2 Jazzy 及编译工具
apt-get update
apt-get install -y --no-install-recommends \
  ros-jazzy-ros-base \
  python3-colcon-common-extensions \
  python3-rosdep

# 6. 安装 ROS2 编译所需的额外依赖包
apt-get install -y --no-install-recommends \
  ros-jazzy-nav2-msgs \
  ros-jazzy-turtlesim \
  ros-jazzy-tf2-ros \
  ros-jazzy-tf2-geometry-msgs \
  ros-jazzy-serial-driver \
  ros-jazzy-ackermann-msgs

# 7. 初始化 rosdep（可选，需联网访问 GitHub）
rosdep init || true
ROSDISTRO_INDEX_URL=https://mirrors.tuna.tsinghua.edu.cn/rosdistro/index-v4.yaml \
  rosdep update --rosdistro jazzy || true

# 8. 配置环境变量自动加载
echo "source /opt/ros/jazzy/setup.bash" >> /etc/bash.bashrc
```

## 7. 兼容性说明

### 7.1 A1 开发板硬件约束

| 参数 | 值 | 编译影响 |
|------|-----|---------|
| CPU | ARM Cortex-A7 @ 1.2GHz | 目标交叉编译为 ARMv7 (armhf) |
| NPU | 0.8 TOPS @ INT8 | 仅支持 ONNX → m1model 格式 |
| 内存 | DDR3L 1Gb (128MB) | 运行时内存紧张，需精简 ROS 节点 |
| 存储 | 256Mb NOR Flash (32MB) | 固件镜像体积需 < 30MB |
| 语言 | C / C++ | 不支持 Python 运行时（板端无 Python） |
| 模型格式 | ONNX（需转换为 m1model） | 通过 SDK 工具链离线转换 |

### 7.2 ROS2 的角色

ROS2 Jazzy **不运行在 A1 开发板上**（板端资源不足）。ROS2 在以下场景使用：

1. **STM32 底盘驱动节点**：在上位机（PC/树莓派）上运行，通过串口（UART 115200）与 STM32 通信
2. **多机导航 / 键盘控制**：在上位机上运行，发布 `cmd_vel` 话题
3. **编译验证**：在 Docker 容器内编译验证 ROS2 包的接口正确性

A1 开发板上运行的是 **纯 C/C++ 程序**（`ssne_ai_demo` / `ssne_vision_demo`），通过 Buildroot SDK 交叉编译。

### 7.3 整体系统架构

```
┌──────────────────────┐    UART    ┌─────────────────────┐
│  上位机 (PC/Pi)       │◄─────────►│  STM32 底盘控制器     │
│  ROS2 Jazzy          │  115200    │  (WHEELTEC C50X)    │
│  + 导航/建图/控制     │   8N1     │  电机/编码器/IMU     │
└──────┬───────────────┘            └─────────────────────┘
       │ USB / Ethernet
       ▼
┌──────────────────────┐
│  A1 视觉处理板       │
│  ARM Cortex-A7       │
│  0.8TOPS NPU         │
│  SC132GS 摄像头      │
│  纯 C/C++ (无 ROS)   │
└──────────────────────┘
```
