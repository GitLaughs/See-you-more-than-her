## 使用说明

### 如何创建 Issue

1. 前往 `https://github.com/GitLaughs/See-you-more-than-her/issues/new`
2. 复制对应 Issue 的标题和描述
3. 添加标签（labels）
4. 设置优先级（Project 或 Milestone）
5. 分配负责人（Assignees）

### Issue 模板

```markdown
**功能描述**:
（简要描述功能或问题）

**技术方案**:
- [ ] 子任务 1
- [ ] 子任务 2
- [ ] 子任务 3

**预期成果**:
（描述完成后的效果）

**相关模块**:
（关联的代码模块或文件）

**备注**:
（其他说明）
```
