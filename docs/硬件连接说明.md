# A1开发板接口信息

## 1. 上层Type-C接口（USB3.0要求）

- **设备类型**：照相机
- **相机参数**：SC132GS，720 x 1280分辨率，90帧/秒，1/4" 灰色Mono，2.1镜头，视场角86.7/49.7/78.9
- **供电方式**：无供电功能，需要开发板供电并通过开关控制
- **电脑连接**：连接电脑后，可用 Aurora 软件监控摄像头拍摄画面（含 OSD 叠加层）
- **伴侣工具**：点云/障碍/检测数据通过 TCP 9090 推送到 [Aurora 伴侣工具](../tools/aurora/README.md) 进行可视化
- **其他软件**：也可以选择电脑端其他软件（如照相机）进行监控，但画面存在明显变色扭曲的问题
- **特殊用途**：此接口将进行Type-C转HDMI扩展坞与无系统显示器连接，查看显示情况
  - **理想情况**：通电后，显示器显示相机实时的拍摄画面
  - **非理想情况**：显示器无显示内容，需要开发板发出控制指令

## 2. 下层Type-C接口（USB2.0）

- **功能**：为开发套件进行供电
- **通信方式**：CH347通信（转换芯片）
- **使用限制**：只可与电脑进行连接，用于编译开发板
- **固件烧录**：通过 CH347 SPI 接口烧录 zImage 固件，详见 [Aurora 伴侣工具 - 烧录](../tools/aurora/README.md#固件烧录)

## 3. GPIO接口（5个）

- **默认状态**：
  - GPIO_PIN_0（左下）：默认状态是UART TX0（输出复用）
  - GPIO_PIN_2：默认状态是UART RX0（输入复用）
- **复用功能**：5个GPIO都有复用功能，通过函数可以设定为UART接口
- **UART参数**：发送/接收 FIFO 均为 32 字节（0x20）
- **使用注意**：uart使用需要注意时序和波特率
- **相关代码**：使用的相关代码查看
