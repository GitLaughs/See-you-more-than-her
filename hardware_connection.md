# A1开发板接口信息
- 上层Type-C接口（USB3.0要求）：
  - 设备为照相机
  - SC132GS	720 x 1280	90	1/4"	灰色Mono	2.1	86.7/49.7/78.9
  - 无供电功能、需要开发板供电并开关控制
  - 连接电脑后，可用aurora软件监控摄像头拍摄画面
  - 同时也可以选择电脑端其他软件（如照相机）进行监控，但画面存在明显变色扭曲的问题。
  - ==此接口将进行Type-C转HDMI扩展坞与无系统显示器连接，查看显示情况==（理想情况：通电，显示器显示相机实时的拍摄画面；非理想情况：显示器无显示内容，需要开发板发出控制指令）
- 下层Type-C接口（USB2.0）：
  - 为开发套件进行供电
  - CH347通信（转换芯片）
  - 只可与电脑进行连接，用于编译开发板
- 5个GPIO接口
  - GPIO_PIN_0（左下）默认状态是UART TX0（输出复用），GPIO_PIN_2默认状态是UART RX0（输入复用）
  - 5个GPIO都有复用功能，通过函数可以设定为UART接口
  - UART 的发送/接收 FIFO 均为 32 字节（0x20）
  - uart使用需要注意时序和波特率
  - 使用的相关代码查看
- 一组I2C接口
  - 不知是否开放
  - 不知位置
  - 根据后续安排查看
  
*sw3开关好像可以控制连接CH347或者连接P4（全部引脚）*
